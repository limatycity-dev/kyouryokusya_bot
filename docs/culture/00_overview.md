# 00_overview — 協力者求むの概要と運用ルール

この文書は、協力者求むの活動・記録・考え方をまとめた
「入口」となるページです。

協力者求むは、日々の活動やコラム、方針、価値観などが
少しずつ積み重なって形づくられていく“文化圏”のようなものです。

そのため、活動の流れや考え方を外部に保存し、
必要なときに参照できる形にしておくことを基本とします。

---

## ■ このアーカイブの目的

1. 活動の文脈を失わないため  
2. 過去の記録やコラムを資産として残すため  
3. 新しく参加する人が状況を理解しやすくするため  
4. 相談や議論の際に、常に同じ前提を共有するため  
5. 協力者求むの“空気”や“流れ”を外部に保つため  

---

## ■ 運用ルール

### 1. Copilot と相談するときは、毎回このアーカイブの URL を貼る  
- URL を貼ることで、その時点の内容を読み込ませる  
- 文脈のズレを防ぎ、話がスムーズになる  

### 2. 記録はすべて md ファイルで管理する  
- フォルダは使わず、番号＋md で並べる  
- 例：  
  - 00_overview.md  
  - 01_manifesto.md  
  - 02_history.md  
  - 03_activity_logs.md  

### 3. 更新は「追記方式」を基本とする  
- 過去の記録は消さず、変化の流れを残す  

### 4. コラム・活動記録・議論ログも保存対象  
- 文章や考え方はすべて“協力者求むの流れ”の一部  

### 5. 相談は必ずアーカイブを前提に行う  
- 「この件について話したい → 該当 md の URL を貼る」  
- Copilot はその文脈を読んで議論する  

---

## ■ この文書の役割

- 協力者求むの入口  
- 活動の扱い方の説明書  
- すべての md の上位に位置する基本ページ  
- 新規参加者が最初に読むべきもの  
- Copilot に文脈を渡すための基準点
